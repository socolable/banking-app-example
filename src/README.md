# 🏦 Secure Modular Banking App

A lightweight, secure, and modular vanilla JavaScript banking application built during a 30-day intensive coding sprint.

## 🌟 Features

- **Secure Transaction Management:** Handle deposits and withdrawals with real-time balance updates.
- **Modular Architecture:** Clean separation of concerns (Logic, UI, Validators, Security).
- **Advanced UI Components:** Custom-built modal system for transaction confirmations.
- **Environment Security:** Integration with `.env` and configuration files to protect sensitive data.
- **Input Validation:** Robust error handling for currency and transaction types.

## 🛠️ Tech Stack

- **Language:** JavaScript (ES6+)
- **Styles:** CSS3 (Flexbox/Grid)
- **Tooling:** Dotenv for environment variables.

## 📂 Project Structure

```text
├── config.js         # Environment & App settings
├── finance.js        # Core banking logic (Calculations)
├── main.js           # App entry point & Event listeners
├── security.js       # Data protection & Encryption logic
├── ui.js             # DOM Manipulation & Modal logic
├── validators.js     # Input sanitization
└── .env              # Private API keys (not committed)
```
